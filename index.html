Choosing a programming language

There are several things to consider when choosing a programming language. This tool can probably help!

What are your objectives in programming?

Select from the following:


Python

Python is a high-level, general-purpose programming language. It is generally considered great for beginners because of its focus on readability and indentation. Python is also popular for in the field of data science. Read more at python.org.

Swift

Swift is a powerful and intuitive programming language for iOS, iPadOS, macOS, tvOS, and watchOS. Writing Swift code is interactive and fun, the syntax is concise yet expressive, and Swift includes modern features developers love. Swift code is safe by design and produces software that runs lightning-fast. Read more about Swift.

Go

Go is a language developed by Google. Go is good for developing cloud applications and microservices because of its many built in toolkits designed for working with the large cloud platforms like AWS, GCP, and Azure. Read more about go at go.dev.

Javascript

JavaScript, is the main programming language used for web development. According to wikipedia, "As of 2022, 98% of websites use JavaScript on the client side for webpage behavior". Anyone doing web development work should learn at least some JavaScript. Read more about JavaScript on MDN.
